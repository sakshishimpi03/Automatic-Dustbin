# Automatic-Dustbin
Technologies Used: Arduino Uno, Ultrasonic Sensor (HC-SR04), Servo Motor, Embedded C, Arduino IDE
Programmed the system in Embedded C using Arduino IDE to detect object distance and control lid movement automatically.
Integrated HC-SR04 ultrasonic sensor for real-time distance measurement and object detection.
Controlled servo motor mechanism to automatically open and close the dustbin lid based on sensor input.
Improved hygiene and user convenience through contactless waste disposal functionality.
